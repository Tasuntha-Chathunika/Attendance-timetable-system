# 🎓 Attendance & Timetable Management System (ATMS)

A comprehensive, full-stack web application designed to streamline student attendance tracking and schedule management. Rebuilt from the ground up for better performance and a modern user experience.

## 🚀 Tech Stack

* **Frontend:** React.js (Vite)
* **Backend:** PHP
* **Database:** MySQL
* **Version Control:** Git & GitHub

## ✨ Key Features (Planned)
* **Secure Authentication:** Role-based login (Admin, Teacher, Student).
* **Timetable Management:** Dynamic creation and viewing of class schedules.
* **Attendance Tracking:** Real-time attendance marking and record keeping.
* **Interactive Dashboard:** Visual analytics for attendance percentages and daily schedules.

## 🛠️ Installation & Setup

### Frontend Setup
1. Clone the repository: `git clone https://github.com/Tasuntha-Chathunika/atms.git`
2. Navigate to the frontend directory: `cd frontend`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

### Backend Setup
1. Move the `backend` folder to your local server directory (e.g., `htdocs` for XAMPP).
2. Import the `database.sql` file into your MySQL database.
3. Update the database connection variables in `config.php`.
